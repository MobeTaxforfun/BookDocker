# Docker Version

安裝後可以用 docker version 確定一下是否成功

docker version

  • Windows docker version

  • Linux docker version

  • 📓 補充 docker version

    其中有一行 Experimental : false
    這是有沒有開起還在實驗中功能的意思
    如果不要開啟可以去 Config 裡面更改設定
    在某個版本之後 Client 是預設開啟的
